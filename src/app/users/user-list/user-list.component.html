Lazy loaded users feature

Liste des users, chargée à la demande.

La route `/users` charge cette feature en lazy loading. Depuis cette liste, chaque carte construit une URL dynamique vers la page dédiée `/users/:userId`. Les users affichés viennent d'un service qui les génère avec Faker, pas d'un tableau hard-codé dans le composant.

Dynamic navigation

Clique une carte : Angular navigue vers une page détail dynamique.

Ici, la liste ne contient aucun outlet. Elle sert de point d'entrée. Le clic redirige vers une page spécifique comme `/users/1`, qui pourra ensuite gérer ses propres nested routes.
